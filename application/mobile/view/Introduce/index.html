4大课程体系介绍

海客机器人课程与STEAM的结合

海客智能机器人课程，致力于为青少年提供智能机器人教育培训解决方案，深度体现STEAM教育理念。通过益智积木搭建，科技模型制作，机器人设计、组装和编程等系列活动，让孩子提升动手能力，学习数理知识，训练逻辑思维，学会分析和解决问题，发展沟通协作技巧等，成为具有独立思考能力和创造力的未来型人才。

MEI课程体系

海客智能机器人课程根据国内外教育现状进行设置，所独创的MEI进阶的课程体系，通过分阶段和针对性，带领处于不同学习阶段的同学进入不同的机器人时代。从低阶到高阶分别为：机械时代、电气时代和智能时代。

  • 机械style
    机械结构与传动学习

    通过不同造型和小球运动的神秘，带给孩子一个奇妙、有趣的游戏性学习氛围，零件安装方便、变化无穷，让孩子在操作的过程充满自主性，独立性和创造性。

  • 电气style
    传感器交互学习

    通过各种工程机械和地面运输工具的构建过程学习，合理结构搭建，了解工程组合，学习安全意识、动手操作、角色扮演，让孩子在学习中充分发挥想象力和创新构建方式，实现提高动手能力的教学目标。

  • 智能style
    智能机器人系统学习

    搭建30个含有机械结构的模型，满足孩子对简单机械构造原理尤其是齿轮运动原理的学习和实现，提高孩子动手构建、理解结构、趣味创新，科学探索的能力。

SPN编程课程体系

海客智能机器人课程为顺应国家政策和时代发展的潮流，独创性地研发了Scratch、Python、NOIP的课程体系。学习编程不仅仅是掌握编程语言，更重要在学习编程的过程中形成编程思维，锻炼思维的逻辑性和完整性，为未来的发展提供更多的可能。

未来是人工智能的时代，更是编程的时代。如今，教育大省北京、山东和浙江都确定把编程纳入信息技术课程和高考的内容体系，学习编程也成为未来的一种趋势 。

  • Scratch
    编程思维训练

    适合年龄：小学、中学

    课程内容：Scratch游戏、动画制作

    Scratch是由麻省理工学院推出的编程工具，是适合于全世界儿童学习编程和交流的工具和平台，该平台采用Java构建，作品可以随意发布到互联网和光盘U盘上。Scratch利用图形化界面，把编程需要的基本技巧囊括其中，包括建模，控制，动画，事件，逻辑，运算，等等。海客学员通过学习Scratch过程中形成编程思维，锻炼思维的逻辑性和完整性，为未来的发展提供更多的可能。
  • Python
    高级动态编程语言学习

    高级动态编程语言学习

    课程内容：Python编程语言学习

    未来是编程的时代，更是Python的时代，Python是人工智能时代第一语言，也是大数据与机器学习的必备语言。海客创造性的研发适合青少年的Python课程 ，通过趣味性的方式，让学员轻松的掌握Python的精华知识。通过Python编程的学习，能够运用Python解决生活中的一些问题。
  • NOIP课程
    C++基础学习、算法学习

    适合年龄：小学、初中、高中

    课程内容：C++语法、算法

    全国青少年信息学奥林匹克联赛（简称NOIP）是中国计算机学会举办的一项竞赛，是教育部认可的五大学科竞赛之一。NOIP主要考察计算机基本组成、操作系统、计算机网络、程序设计知识、数学逻辑知识等，要求学生有较强的编程和上级调试的实践能力。NOIP从2020年起取消PASCAL语言和C语言试题，所以本课程使用C++授课。

机器人竞赛

海客建立了一套完整的机器人竞赛培训体系，为竞赛项目设置独立的部门。在金牌教练的指导下，在科协和电教馆组织的全国、省、市各级机器人官方比赛，VEX、 WER等国际官方赛事中均屡屡斩获大奖，获奖学员不仅在升学和自主招生考试中获得降分资格，在报考世界名校中也能获得更多优势。

  • VEX工程挑战赛

    课程内容: VEX EDR、VEX IQ

    VEX是全球规模最大，最受科技类大学认可的机器人比赛，也是中国青少年机器人竞赛中的重要项目。

  • VER工程挑战赛

    课程内容:机械搭建、机器人编程

    WER是一项面向全球4-18岁青少年的教育机器，人比赛，每年有超过50多个国家的50万名选手参加各级WER选拔赛。

  • 综合技能

    课程内容:机械搭建、机器人编程

    综合技能比赛是中国青少年机器人竞赛项目之一，参赛需代表现场自行拼装、编制运行程序、调试和操作机器人。

  • 机器人创意

    课程内容:传感器交互、机器人编程

    机器人创意培养综合运用机器人技术、电子信息技术、工程技术，激发创新思维潜能，提高综合设计和制作的能力。

  • Make X

    课程内容:机械搭建、机器人编程

    MakeX是一个引导青少年成长的机器人赛事平台，旨在通过机器人挑战赛、激发大众对创造、探索、分享的热情。

机器人幼儿课程体系

海客智能机器人幼儿课程理念源于STEAM教育，专注于3-6岁幼儿的机器人课程。以“玩中学”、“恰当的任务”、“ 快乐学习”的教学理念致力于培养中国孩子创新思维、创新能力、科学素养等综合实力。

  • 幼儿园小班课程

    通过不同造型和小球运动的神秘，带给孩子一个奇妙、有趣的游戏性学习氛围，零件安装方便、变化无穷，让孩子在操作的过程充满自主性，独立性和创造性。

  • 幼儿园中班课程

    通过各种工程机械和地面运输工具的构建过程学习，合理结构搭建，了解工程组合，学习安全意识、动手操作、角色扮演，让孩子在学习中充分发挥想象力和创新构建方式，提高动手能力。

  • 幼儿园大班课程

    搭建30个含有机械结构的模型，满足孩子对简单机械构造原理尤其是齿轮运动原理的学习和实现，提高孩子动手构建、理解结构、趣味创新，科学探索的能力。
